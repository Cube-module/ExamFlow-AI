# ExamFlow AI

Telegram-бот для структурированной подготовки к ЕГЭ/ОГЭ с ИИ-помощником и системой геймификации.

## О проекте

ExamFlow AI решает три главных барьера при подготовке к экзаменам:

- **Структурный** — готовые курсы с логичной навигацией по темам
- **Понятийный** — ИИ объясняет сложные моменты прямо в чате, когда это нужно
- **Мотивационный** — серии дней, достижения и напоминания удерживают дисциплину

ИИ не создаёт курс — он усиливает прохождение готового материала: объясняет, генерирует задачи, проверяет ответы.

## Возможности

### Курсы и навигация
- Карта курса с прогрессом (пройдено / в процессе / заблокировано)
- Переход к следующему уроку только после выполнения минимальных условий
- Возможность вернуться к любому пройденному материалу

### ИИ-помощник
- **«Не понял»** — упрощённое объяснение с примерами и аналогиями
- **«Хочу практику»** — генерация задач по теме с нарастающей сложностью
- **Проверка ответов** — сверка с эталоном или оценка развёрнутых решений через LLM

### Геймификация
- **Серия дней (Streak)** — растёт при ежедневном выполнении нормы (1 урок + 5 задач)
- **Достижения** — бейджи за постоянство, мастерство и преодоление сложных тем
- **Заморозка** — защита серии от сгорания, получается за достижения
- **Напоминания** — уведомление за 4 часа до конца суток, если норма не выполнена

### Личный кабинет
- Текущая серия дней
- Процент прохождения по модулям
- Статистика правильных/неправильных ответов
- Рекомендации по слабым темам

## Стек

| Компонент | Технология |
|---|---|
| Bot Framework | aiogram 3.x |
| LLM | Groq API (Llama 3.3 70B) |
| База данных | SQLite (dev) / PostgreSQL (prod) |
| Планировщик | APScheduler |

## Установка и запуск

```bash
# Клонировать репозиторий
git clone <repo-url>
cd ExamFlow-AI

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env
# Заполнить TELEGRAM_BOT_TOKEN и GROQ_API_KEY

# Запустить бота
python bot/main.py
```

## Переменные окружения

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
GROQ_API_KEY=your_groq_api_key
```

## Как пользоваться ботом

1. Найди бота в Telegram и отправь `/start`
2. Выбери курс (например, «Математика ЕГЭ профиль»)
3. Изучай уроки последовательно:
   - **«Понял, дальше»** — переход к следующему уроку
   - **«Не понял, объясни»** — ИИ даст дополнительное объяснение
   - **«Хочу практику»** — ИИ сгенерирует задачи по теме
4. Заходи каждый день, чтобы поддерживать серию
5. Следи за прогрессом в разделе «Профиль»
